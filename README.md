# 📜 AstrBot 对话 Token 使用报告

> 用于 [AstrBot](https://github.com/AstrBotDevs/AstrBot) 的轻量级 Token 额度审计插件。支持按轮次、时间间隔自动统计模型消耗，提供详尽的文本模型 Token 使用报告。

## ✨ 核心特性

- **📊 深度统计**：实时监控文本模型的 Token 使用情况（包括输入、输出及总计）。
- **⚙️ 灵活触发机制**：
  - **条件设置**：可自由设定报告触发条件，支持“达到问答轮次”、“固定时间间隔”或“两者同时生效”。
  - **环境隔离**：群聊与私聊支持独立配置，按需开关特定窗口的记录功能，确保只统计您关心的频道。
- **📈 自动化报告**：无需手动干预，达到设定阈值后自动推送结构化统计报告。

## 🚀 安装方法

1. 通过 Release 获取最新版本的 `.zip` 格式插件包。
2. 在 AstrBot WebUI 的 **插件安装** 页面上传安装即可生效。

## 🛠️ 使用方法

### 自动报告
在配置文件或 WebUI 设置面板中，根据您的需求设定触发轮次或时间间隔，即可自动为指定的窗口发送报告。

## 📋 报告示例

插件运行后生成的报告效果如下：

```text
📊token使用报告[XX]
🏷️[1XXXXXXXX8][私聊]
      期间有效对话轮次：2次
      期间模型调用总次数：2次
      期间Token使用总量：11.7k [输入11.7k/输出21/其他0]
      调用模型名称
           -gpt-5.4-nano[语言]：
               [输入11.7k/输出21/其他0/合计11.7k/调用次数2/平均调用5860/最高单次5992]
```
